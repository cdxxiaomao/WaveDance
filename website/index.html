WaveDance 采集系统播放音频，实时绘制丝滑频谱浮层。适用于直播、创作、专注与沉浸式背景展示。

核心特性

实时频谱渲染

基于 WebGL 实时绘制系统音频频谱，保证流畅视觉反馈与低延迟响应。

高度可调

支持分桶、补偿、频率区间、显示模式等参数调整，快速调出你想要的动态效果。

原生桌面体验

通过 Tauri 打包，提供轻量、稳定的 macOS 桌面体验，并可输出 `.dmg` 发布包。

三步完成安装

1

安装应用

下载并打开 `.dmg`，将 WaveDance 拖入「应用程序」后启动。

2

配置 BlackHole

在设置中安装或检测 BlackHole 2ch，用于系统音频回环采集。

3

授权并开始采集

允许麦克风权限后播放任意音频，观察频谱随声音实时变化。

详细图文指南可查看仓库文档 `docs/INSTALL.md`。

常见问题

为什么没有频谱变化？

请确认系统输出已切换到 BlackHole（或多输出设备），并检查麦克风权限是否授予 WaveDance。

支持哪些系统版本？

当前面向 macOS 12 及以上版本。

是否可以从源码运行？

可以，仓库 `README.md` 提供了前端依赖安装与 `cargo tauri dev` 的开发流程。
