BlackHole 随包安装说明
====================

本应用只会用 macOS 的「安装器」打开 .pkg 文件，不能安装「仅有源码的文件夹」。

请从官方 Release 下载带 .pkg 的安装包（名称通常类似 BlackHole2ch-0.6.1.pkg），
可任选其一放置方式：

  • resources/blackhole/ 下任意 .pkg
  • resources/blackhole/某子文件夹/ 内任意 .pkg（例如你解压后的一层目录里）
  • 或仍命名为 resources/blackhole/BlackHole.pkg

若目录里没有任何 .pkg（例如只有 GitHub 的源码包 BlackHole-0.6.1/），
应用无法代为安装，请到 Release 页下载 .pkg：

  https://github.com/ExistentialAudio/BlackHole/releases

许可证以 BlackHole 官方为准；随产品分发时请保留第三方版权声明。
