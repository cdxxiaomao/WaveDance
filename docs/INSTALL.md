# WaveDance 安装说明

本文面向**最终用户**，说明在 macOS 上安装 WaveDance、安装 BlackHole 虚拟声卡，以及完成系统声音路由与权限配置。

---

## 一、系统要求

- **macOS 12（Monterey）或更高版本**
- 管理员账户（安装 BlackHole 时系统会要求输入密码）
- 若从网络下载应用，首次打开可能需在「系统设置 → 隐私与安全性」中允许运行

---

## 二、安装 WaveDance

1. 打开发行方提供的 **`WaveDance.dmg`**（文件名可能带版本与架构，例如 `WaveDance_0.1.0_aarch64.dmg`）。
2. 将窗口中的 **`WaveDance`** 图标拖入 **「应用程序」** 文件夹。
3. 推出磁盘映像（点击桌面上的 DMG 图标，右键「推出」）。
4. 在 **「启动台」** 或 **「应用程序」** 中打开 **WaveDance**。

若首次打开提示「无法打开，因为来自身份不明的开发者」：

1. 打开 **「系统设置」→「隐私与安全性」**。
2. 在「安全性」相关区域找到被拦截的提示，选择 **「仍要打开」**（具体文案随 macOS 版本略有不同）。

---

## 三、安装 BlackHole（推荐）

WaveDance 通过 **BlackHole** 这类虚拟音频设备，把「系统正在播放的声音」作为输入来绘制频谱。需要先安装 BlackHole，再把系统输出指到该设备。

### 方式 A：在应用内安装（推荐）

1. 打开 WaveDance，进入 **「设置」**（按界面上的设置入口打开设置窗口）。
2. 在 **「系统音频（BlackHole）」** 区域查看检测结果。
3. 若提示未检测到 BlackHole，点击 **「安装 BlackHole…」**。
4. 在弹出的确认框中选 **「确定」**。
5. 系统将打开 **BlackHole 安装程序**（若发行包内已附带安装包）；否则会打开 **官方下载页面**，请下载 **2 通道（2ch）** 的 `.pkg` 安装包并双击安装。
6. 按安装器提示操作；需要时 **输入本机管理员密码**，并在 **「系统设置」** 中按提示允许与音频相关的组件（若系统弹出）。

安装完成后，建议在设置中点击 **「重新检测」**，确认文案变为已检测到 BlackHole。

### 方式 B：自行从官网安装

若你更习惯手动安装：

1. 打开：<https://github.com/ExistentialAudio/BlackHole/releases>
2. 下载 **BlackHole 2ch** 对应的 **`.pkg`**（不要下载 Source code 源码压缩包）。
3. 双击 `.pkg`，按向导完成安装。

---

## 四、把系统声音输出到 BlackHole

仅安装 BlackHole 还不够，需要让 **系统正在播放的声音** 经过 BlackHole，WaveDance 才能采到信号。

1. 打开 **「系统设置」→「声音」**（或旧版「系统偏好设置」→「声音」）。
2. 在 **「输出」** 中选中 **「BlackHole 2ch」**（名称以实际安装为准）。

**注意：** 输出选成 BlackHole 后，声音可能**不会从 Mac 扬声器播出**（取决于你的多设备配置）。若希望 **扬声器仍能听到** 且同时给 WaveDance 回环，可使用 macOS **「音频 MIDI 设置」** 中的 **「多输出设备」**，将扬声器与 BlackHole 组合为同一输出；具体步骤因系统版本而异，可自行搜索「macOS 多输出设备 BlackHole」。

---

## 五、麦克风权限（首次采集时）

WaveDance 使用系统提供的音频输入接口读取 BlackHole 等设备上的音频流；macOS 会将其归类为 **麦克风** 类权限。

1. 在应用中 **开始采集** 或首次需要音频时，若弹出权限请求，请选择 **「好」** 或 **「允许」**。
2. 若曾误点拒绝，可到 **「系统设置」→「隐私与安全性」→「麦克风」** 中，勾选 **WaveDance**。

若仍无波形，可在「终端」中重置该应用的麦克风授权后重启应用（包名以实际为准）：

```bash
tccutil reset Microphone com.wavedance.desktop
```

---

## 六、验证是否安装成功

1. 系统输出已指向 BlackHole（或已配置多输出）。
2. 用音乐或视频软件 **正常播放有声音的内容**。
3. 打开 WaveDance 设置，**开始采集**，观察频谱是否随声音变化。

若采集已启动但无变化：先确认 **输出设备** 与 **BlackHole 是否已安装并被检测到**，再检查 **麦克风权限**。

---

## 七、常见问题

| 现象 | 建议 |
|------|------|
| 设置里一直显示未检测到 BlackHole | 确认已用 `.pkg` 完成安装；安装后点「重新检测」；必要时注销或重启后再试。 |
| 安装器打不开 | 在「隐私与安全性」中允许；或从官网重新下载 `.pkg`。 |
| 有权限仍无波形 | 确认当前播放声音确实从「输出到 BlackHole」的路径经过；尝试停止再重新开始采集。 |
| 只想开发/自己编译 | 见仓库根目录 `README.md` 与 `BUILD_MACOS.md`。 |
